File Descriptions

0-strcat.c: Concatenates two strings.

1-strncat.c: Concatenates two strings, to a maximum of n characters from src.

2-strncpy.c: Copies a string, to a maximum of n characters from src.

3-strcmp.c: Compares two strings.

4-rev_array.c: Reverses the content of an array of n integers

5-string_toupper.c: Converts all lowercase letters in a string to uppercase.

6-cap_string.c: Capitalizes all words on a string.
